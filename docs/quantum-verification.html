Why Neural Cryptanalysis?

Neural networks can detect subtle statistical patterns that traditional tests miss. Gohr (CRYPTO 2019) demonstrated that deep learning can break reduced-round Speck32/64. If a neural network trained specifically to find distinguishing patterns achieves only random-guessing accuracy, it provides strong empirical evidence of SPRP security.

Simon's Algorithm — The Core Threat

Simon's algorithm finds a hidden period s where f(x) = f(x ⊕ s) in O(n) quantum queries — an exponential speedup over classical. In Feistel networks, one half passes through unchanged each round, creating the algebraic relationships Simon's algorithm exploits.

FAST's SPN eliminates this by transforming the entire block every round. Test 3 verifies no period exists (bijection on 10K values), and Test 7 confirms no Feistel-like half-block preservation.
