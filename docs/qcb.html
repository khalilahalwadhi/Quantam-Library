QCB-AEAD

Quantum Ciphertext Block — Provably Q2-Secure Authenticated Encryption

ASIACRYPT 2021 · IETF RFC 9771 · Syncoda / Khalilah Aisha al-Wadhi

Read the Whitepaper API Documentation

The Problem

Every widely deployed AEAD mode — GCM, CCM, EAX, OCB, GCM-SIV — relies on algebraic structures that are broken in polynomial time by Simon's quantum period-finding algorithm when an adversary can issue superposition queries.

GCM

Broken. GHASH polynomial multiplication creates an exploitable period in the universal hash function.

OCB

Broken. Gray-code counter structure provides the exact periodicity Simon's algorithm needs.

GCM-SIV

Broken. POLYVAL shares the same polynomial vulnerability as GHASH.

QCB

Secure. XEX-based TBC eliminates all exploitable periodicity. Proven Q2-secure at ASIACRYPT 2021.

How QCB Works

Plaintext:  M₁    M₂    M₃  ···  Mₘ
             │      │      │        │
             ▼      ▼      ▼        ▼
           ┌────┐ ┌────┐ ┌────┐  ┌────┐
           │Ẽ_K │ │Ẽ_K │ │Ẽ_K │  │Ẽ_K │  ← Tweak = (domain, nonce, index)
           └────┘ └────┘ └────┘  └────┘
             │      │      │        │
             ▼      ▼      ▼        ▼
Ciphertext: C₁    C₂    C₃  ···  Cₘ

Checksum = M₁ ⊕ M₂ ⊕ ··· ⊕ Mₘ
Tag = Ẽ_K(0x05‖N‖0, Checksum ⊕ AD_hash)

XEX Tweakable Block Cipher

Ẽ_K(T, X) = AES_K(X ⊕ Δ) ⊕ Δ    where Δ = AES_K(T)

Each tweak mask Δ is derived through a full AES encryption, eliminating the algebraic periodicity that Simon's algorithm exploits in other modes.

Quick Start

Rust

use qcb_core::{Qcb, QcbKey};

let key = QcbKey::new(&[0x42; 32]).unwrap();
let cipher = Qcb::new(&key);

let ct = cipher.encrypt(
    &[0u8; 12],       // nonce
    b"associated data", // AAD
    b"Hello, quantum-safe world!",
).unwrap();

let pt = cipher.decrypt(&[0u8; 12], b"associated data", &ct).unwrap();
assert_eq!(pt, b"Hello, quantum-safe world!");

Python

from qcb_aead import QcbCipher

cipher = QcbCipher(key=b"\x42" * 32)
ct = cipher.encrypt(nonce=b"\x00" * 12, data=b"Hello!", aad=b"metadata")
pt = cipher.decrypt(nonce=b"\x00" * 12, ciphertext=ct, aad=b"metadata")
